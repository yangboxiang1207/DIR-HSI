# DIR-HSI
DIR-HSI: Degradation-Invariant Representation Learning for Hyperspectral Image Classification
Please create a new 'dataset' folder. Inside it, create a 'PaviaU' subfolder to store the PU data (two .mat files), and also create a 'noise_Pavia' subfolder to store the degradation data (.mat files).
Please refer to HyperTTA (https://github.com/chenning0115/HyperTTA) for generating the relevant degradation files.
